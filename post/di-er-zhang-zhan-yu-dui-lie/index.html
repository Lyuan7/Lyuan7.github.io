第二章-栈与队列

2019-12-01 Views 3 min read             繁
 

基本概念

  1. 栈的定义：栈是一种只能在一端进行插入和删除操作的线性表。允许进行插入和删除操作的一端称为栈顶（Top）而栈底时固定不变的
  2. 栈的特点：先进后出
  3. 栈的存储结构：顺序栈&链栈（这里只着重说顺序栈）
  4. 队列的定义：队列仅允许在表的一端进行插入，在表的另一端进行删除
  5. 队列的特点: 先进先出

存储结构&算法

栈的存储结构设计

栈的初始化：建立一个空栈
判断栈空,，取栈顶元素, 入栈，出栈
具体的各项操作如下

//定义栈的结构
typedef struct Stack{
    int stack[maxsize]; //存放栈的元素, maxsize是已经定义的常量
    int top; //栈顶指针
}SeqStack;

//初始化栈
void init_seqstack(SeqStack *s){
    s->top = -1; //这里规定-1为栈空, 因为0会因为预留一个空间导致浪费
}

//入栈操作
int push_seqstack(SeqStack *s, int x){
    if(s->top == maxsize) //判断栈满
        return FALSE;
    else{
        s->top++;
        s->stack[s->top] = x;
    }
    return true;
}
 
//出栈操作
int pop_seqstack(SeqStack *s, int *x){ //因为需要输出出栈元素值,所以使用指针类型
    if(s->top == -1) //判断栈空
        return FALSE;
    esle{
        *x = s->stack[s->top];
        s->top--;
    }
    return True;
}

队的存储结构设计

//顺序队列的定义
typedef struct{
    int data[maxsize];  //存储数据
    int front; //头指针
    int rear;  //尾指针
}SqQueue;

//初始化队列
void init_Queue(SqQueue * qu){
    qu->front = qu->rear = 0;  //队首和队尾指针重合,并且指向0
}

//入队算法
int enQueue(SqQueue *qu, int x){
    if((qu->rear+1)%maxsize == qu.front)  //判断队满
        return 0;
    qu->rear = (qu->rear+1)%maxsize;  //先移动指针,再存入数据
    qu->data[qu->rear] = x;
    return 1;
}

//出队算法
int deQueue(SqQueue *qu, int *x){
    if(qu->front == qu->rear) //队空,则不出队
        return 0;
    qu->front = (qu->front+1)%maxsize;
    x= qu->data[qu->front];
    return 1;
}

循环队列详解

先来考虑一种情况
因为队列的特殊性, 只能够在一端插入, 在另一端删除. 所以随着入队及出队运算的不断进行,
会出现一种情形: 队列在数组中不断地向队尾方向移动, 而在队首的前面则产生了一片不能利用的空闲存储区.

当尾指针指向数组的最后一个位置时(rear = maxsize -1) 就不能够再加入新元素. 这种现象称为假溢出
当假溢出时, 新元素不能够入队. 所以我们应该想出以一种首位相接的环形区域来表示队列: 循环队列
